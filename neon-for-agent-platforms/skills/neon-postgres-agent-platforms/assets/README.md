# Assets

Optional **`assets/`** files (templates, static data, images) load only when needed, see **[Agent Skills specification, assets](https://agentskills.io/specification#assets)**.

This companion skill does not ship binary templates today; procedural guidance lives in **`SKILL.md`** and **`references/`**. For JSON/env patterns, see **[`examples/api-scripts/.env.example`](../../../examples/api-scripts/.env.example)** in the repo root.
